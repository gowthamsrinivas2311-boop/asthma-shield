# Asthma Shield

Asthma Shield is a tool designed to help asthma patients understand the impact of air pollution.

## Features
- Converts AQI into cigarette-equivalent exposure based on Berkeley Earth PM2.5 equivalency research
- Helps users interpret pollution levels 
- Users can log symptoms and analyse patterns
- Provides asthma-related guidance
- Patients can practice their breathing through guided gamified breathwork
- Can call emergency services and hospitals by just a click eliminating time waste in critical situations
